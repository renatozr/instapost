## Getting Started

### Prerequisites

- NPM

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:renatozr/instapost.git
   ```
2. Enter the directory
   ```sh
   cd instapost
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm run dev
   ```
